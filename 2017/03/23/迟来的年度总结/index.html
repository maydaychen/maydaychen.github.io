迟来的年度总结

时间真的像流水，匆匆越过指尖，再怎么使劲也抓不住一点。珍惜现在，立足当前，显然这些话已经是老生常谈了。从去年六月份毕业以来，感觉一开始像是开足了马力的拖拉机，学会了好多，但是始终是一个人在做着app开发，学什么都感觉有点跑偏。从年初到现在，活也没有了，感觉就像一条咸鱼，在慢慢风干。这种心情我也不知道怎么描述，内心暗自庆幸可以有更多自由的时间来研究新的知识，又觉得这样很闲很无聊的生活并不是我想要的。温水煮青蛙，慢慢的，我也不知道会不会成为一个废人。每天逛逛简书，鸡汤也好，新闻也罢，亦或是听着熟悉还证明我没有忘记基础的技术知识也好，想着当年说，年轻嘛，就该多闯闯，怎么就成了这个样子……

好了，该抱怨的也抱怨过了，接下来，就要写自己给自己的鸡汤了。17年，是一个很重要的一年，基本上，将来废不废，就靠这一年了，毕业工作后的第一年，就算环境再不好，也要做最好的自己不是么？去年听了一年的这货

如今经过三个月的试水探索，发现还真有这迹象╮(╯▽╰)╭，毕竟经验少，要的全是3-5年的高手，日子不好混啊。今年打算略微接触一下前端，至少搞懂一下RN的相关方法。其次，继续巩固Android知识，动画也好，网络也好，还有一些其他领域的知识都尝试一下，多逛逛GitHub。

#2017书单总结

  1. Android群英传
  2. Android开发艺术探索
  3. Android应用性能优化
  4. (备选)Effective Java

#其他

  1. 多多锻炼吧，下了个quick Fit，每天照着锻炼下，感觉应该是有用的
  2. 我想买个surface 键盘/(ㄒoㄒ)/~~但是不是机械键盘要八百，软爸爸你真的不是打劫么？

#总结
2017，希望万事顺利吧，入了社会，才知道真的是深似海，在学校被保护着没发现，现在发现当年的希望早日毕业的我是多么纯真幼稚。
