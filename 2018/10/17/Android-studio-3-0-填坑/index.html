Android studio 3.0 填坑

发表于 2018-10-17 | 分类于 Android

#1.依赖项报错：

解决方案：
将compile project(‘:wheel’)
改为
compile project(path: ‘:easeUI’, configuration: ‘default’)

#2.retrolambda报错：

解决办法：
将
module层gradle中的apply plugin: ‘me.tatarka.retrolambda’
及project层gradle中的classpath ‘me.tatarka:gradle-retrolambda:3.2.5’
注释掉

#3.apt插件报错

解决方案：
1.在project的build.gradle中删除
classpath ‘com.neenbedankt.gradle.plugins:android-apt:1.8’
2.在module的build.gradle中删除
apply plugin: ‘android-apt’
3.将module的build.gradle文件中的dependency
中所有的apt改为annotationProcessor
