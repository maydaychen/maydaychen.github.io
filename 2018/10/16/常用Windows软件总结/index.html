常用Windows软件总结

发表于 2018-10-16 | 更新于 2018-10-17 | 分类于 others

为了每次重装电脑的时候不会遗漏相关软件，特写下此文章，提醒自己保持较高生产力

#图片查看

  1. Picasa 3
    出师不利……貌似谷歌爸爸不支持了

#服务器相关

  1. Xshell
    远程连接服务器，执行相关操作
    https://www.netsarang.com/products/xsh_overview.html
  2. FileZilla
    服务器远程文件传输
    https://filezilla-project.org/

#影音

  1. Listen1
    支持网易云音乐/QQ音乐/虾米音乐三合一
    https://github.com/listen1/listen1_chrome_extension

  2. 网易云音乐
    以前很喜欢，可是现在- -版权实在太少，能听的不多，而且推荐算法工程师可能换人了……
    https://music.163.com/

  3. win10商店中的爱奇艺
    没！有！广！告！

#工具

  1. 微软 To- Do
    每日备忘记录提醒
    https://todo.microsoft.com/zh-cn

  2. geek
    很舒服的卸载工具
    https://geekuninstaller.com/

  3. TeamViewer
    远程控制软件，work from home 必备
    https://www.teamviewer.com/zhCN/

  4. notepad++
    替代记事本良心直选
    https://notepad-plus-plus.org/
